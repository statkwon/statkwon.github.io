1. Introduction to Bokeh

from bokeh.io import output_notebook, show
from bokeh.plotting import figure

우선 위의 명령어를 사용하여 Bokeh를 import한다.

output_notebook()

기본적으로 Bokeh는 html 형식의 결과물을 산출하지만, output_notebook() 명령어를 입력함으로써 결과물을 Jupyter Notebook에서 확인할 수 있다. html 형식의 결과물을 저장하고 싶다면 bokeh.io에서 output_file을 호출한 후 output_file(‘~.html’) 명령어를 입력하면 된다.

plot = figure()
show(plot)
Bokeh Plot

Bokeh는 항상 figure() 명령어로 빈 프레임을 만들어주는 것으로 시작한다.

1. Markers

plot = figure(plot_width=400, plot_height=400, tools='pan, reset, save')
plot.circle([1, 2, 3, 4], [1, 2, 3, 4])
show(plot)
Bokeh Plot

plot의 크기나 툴바 등을 조정하고 싶은 경우에는 figure() 안에 plot_width, plot_height, tools 등의 옵션을 추가해야 한다. 이 외에도 x축, y축 라벨이나 전체 제목 역시 figure() 안에서 설정할 수 있다. 완성된 빈 프레임에 점을 추가하고 싶다면 figure 뒤에 .circle 명령어를 입력하면 된다. 추가적으로 color나 size 등의 옵션을 통해 점의 크기와 색상을 조정할 수 있다.

plot = figure(plot_width=400, plot_height=400)
plot.x([1, 2, 3, 4], [1, 2, 3, 4], color='red', size=10)
show(plot)
Bokeh Plot

점의 모양을 바꾸고 싶은 경우에는 .circle 대신 .x 등의 다른 명령어를 사용한다. Bokeh에서 사용할 수 있는 Marker의 종류에는 asterisk, circle, circle_cross, circle_x, cross, diamond, diamond_cross, inverted_triangle, square, square_cross, square_x, triangle, x 등이 있다.

4. Histogram

import numpy as np
plot = figure(plot_width=400, plot_height=400, x_axis_label='Sepal Length', y_axis_label='Probability',
             title='Histogram')
hist, edges = np.histogram(flowers['sepal_length'], density=True, bins=10)
plot.quad(top=hist, bottom=0, left=edges[:-1], right=edges[1:], line_color="white", color='orange')
show(plot)
Bokeh Plot

Bokeh에서는 히스토그램을 그리는 기능을 별도로 지원해주지 않는다. 그렇기 때문에 Bokeh 화면 상에 히스토그램을 띄우고 싶다면 numpy의 np.histogram을 통해 히스토그램의 형태를 만든 후, .quad 명령어를 사용하여 여러 개의 직사각형들을 추가하는 방법을 사용해야 한다. .quad() 안에 반드시 히스토그램의 상, 하, 좌, 우 라인의 값을 지정해주어야 한다.

5. Hover Appearance

그래프 위에 마우스를 올렸을 때 색이 바뀌거나 특정한 정보를 보이게 하는 것이 가능하다. 이러한 기능을 ‘Hover’라고 한다.

from bokeh.models import HoverTool

Hover 기능을 사용하려면 우선 bokeh.models로부터 HoverTool을 import 해야한다.

hover = HoverTool(tooltips=None, mode='vline')
plot = figure(plot_width=400, plot_height=400, tools=[hover],
              x_axis_label='Petal Length', y_axis_label='Sepal Length', title='Petal Length vs. Sepal Length')
plot.circle(flowers['petal_length'], flowers['sepal_length'], size=5,
            hover_fill_color='firebrick', hover_alpha=0.5, hover_line_color='white')
show(plot)
Bokeh Plot

HoverTool() 안에서 mode 옵션을 통해 Hover의 방향을 설정할 수 있다. ‘vline’을 입력하면 수직으로, ‘hline’을 입력하면 수평으로 기능이 적용 된다. tooltips에도 값을 입력할 수 있지만, 우선은 None으로 설정하도록 하겠다. HoverTool()을 하나의 변수에 할당하였다면, 반드시 figure() 내의 tools 옵션에 해당 변수를 추가해주어야 한다. 여기까지 했다면 기본적인 Hover 기능이 구현될 것이다. 색상이나 투명도 등 세부적인 옵션을 조절하고 싶다면 .circle이나 .line 등 오브젝트와 관련된 명령어 안에 hover_fill_color, hover_alpha, hover_line_color 등의 옵션을 추가해주면 된다.

from bokeh.models import ColumnDataSource

이번에는 마우스를 올렸을 때 특정 정보가 보이도록 하는 Hover 기능을 만들어 보자. 이 작업을 하기 위해서는 우선 bokeh.models로부터 ColumnDataSource를 import 해야한다.

source = ColumnDataSource(flowers)
hover = HoverTool(tooltips=[('Species', '@species'), ('Petal Length', '@petal_length'),
                            ('Sepal Length', '@sepal_length')])
plot = figure(plot_width=400, plot_height=400, tools=[hover],
              x_axis_label='Petal Length', y_axis_label='Sepal Length', title='Petal Length vs. Sepal Length')
plot.circle('petal_length', 'sepal_length', source=source, size=5)
show(plot)
Bokeh Plot

ColumnDataSource() 안에 사용할 데이터프레임을 집어 넣고 source라는 변수로 지정한 후, figure() 안의 source 옵션으로 해당 변수를 입력해주면 컬럼명만 가지고 작업하는 것이 가능해진다. 이러한 편리를 위해서 뿐만 아니라, 우리가 만들고자 하는 정보 pop-up 기능을 사용하려면 ColumnDataSource를 필수적으로 사용해야 한다. 이제 위에서 비워두었던 HoverTool의 tooltips를 수정해줘야 한다. tooltips에 [(‘입력하고 싶은 텍스트’, ‘@컬럼명’)] 형태의 리스트를 추가하게 되면, ‘입력하고 싶은 텍스트: 해당 컬럼의 값’ 형태로 정보가 나타나게 된다.

6. Color Mapping

from bokeh.models import CategoricalColorMapper
plot = figure(plot_width=400, plot_height=400, x_axis_label='Petal Length', y_axis_label='Sepal Length',
             title='Petal Length vs. Sepal Length')
source = ColumnDataSource(flowers)
mapper = CategoricalColorMapper(factors=['setosa', 'virginica', 'versicolor'], palette=['red', 'green', 'blue'])
plot.circle('petal_length', 'sepal_length', color={'field':'species', 'transform':mapper}, source=source)
show(plot)
Bokeh Plot

다음은 범주에 따라 색상을 다르게 표현하는 방법에 대해서 알아도록 하자. Seaborn에서는 ‘hue’ 옵션을 추가함으로써 쉽게 구현할 수 있는 기능이지만, Bokeh에서는 상대적으로 더 복잡한 과정이 필요하다. 이 기능 역시 우선 ColumnDataSource에 데이터프레임을 할당해주어야 사용할 수 있다. 해당 작업을 마쳤다면, CategoricalColorMapper()라는 명령어 안에 factors와 palette 두 가지 옵션을 설정해주어야 한다. factors에는 색상 구분의 기준이 될 범주를, palette에서는 범주별 색상을 입력하면 된다. 이후 .circle() 등 Marker 관련 명령어의 color 옵션에 {‘field’:‘species’, ‘transform’:mapper} 형태로 값을 입력해주면 된다. 이때 ‘field’에는 색상을 구분할 범주가 되는 변수를, ‘transform’에는 앞서 만들었던 CategoricalColorMapper를 할당하면 된다.

plot = figure(plot_width=400, plot_height=400, x_axis_label='Petal Length', y_axis_label='Sepal Length',
             title='Petal Length vs. Sepal Length')
source = ColumnDataSource(flowers)
mapper = CategoricalColorMapper(factors=['setosa', 'virginica', 'versicolor'], palette=['red', 'green', 'blue'])
plot.circle('petal_length', 'sepal_length', color={'field':'species', 'transform':mapper},
            legend_field='species', source=source)
plot.legend.location = 'top_left'
show(plot)
Bokeh Plot

추가적으로 legend_filed 옵션에 색상 구분의 범주가 되는 변수의 이름을 할당하여 범례를 생성하는 것도 가능하다.

plot = figure(plot_width=400, plot_height=400, x_axis_label='Petal Length', y_axis_label='Sepal Length',
             title='Petal Length vs. Sepal Length')
source = ColumnDataSource(flowers)
mapper = CategoricalColorMapper(factors=['setosa', 'virginica', 'versicolor'], palette=['red', 'green', 'blue'])
plot.circle('petal_length', 'sepal_length', color={'field':'species', 'transform':mapper},
            legend_field='species', source=source)
plot.legend.location = 'top_left'
plot.legend.background_fill_color = 'lightgray'
show(plot)
Bokeh Plot

plot.legend.location으로 범례의 위치를 바꾸거나, plot.legend.background_fill_color로 범례의 배경색을 변경할 수 있다.

2. Arrange Plots

이번에는 여러 개의 plot을 동시에 나타내는 방법에 대해 알아보도록 하자.

1. Row of Plots

from bokeh.layouts import row

가로로 plot들을 배열하고 싶다면 bokeh.layouts으로부터 row를 import 해야한다.

source = ColumnDataSource(flowers)

p1 = figure(plot_width=400, plot_height=400, x_axis_label='Petal Length', y_axis_label='Sepal Length',
           title='Petal Length vs. Sepal Length')
p1.circle('petal_length', 'sepal_length', source=source, color='red')

p2 = figure(plot_width=400, plot_height=400, x_axis_label='Petal Length', y_axis_label='Petal Width',
           title='Petal Length vs. Petal Width')
p2.circle('petal_length', 'petal_width', source=source, color='blue')

layout = row(p1, p2)
show(layout)
Bokeh Plot

우선 화면에 나타내고 싶은 plot들을 각각 만들어준 다음, row() 안에 추가해주기만 하면 가로 배열을 완성할 수 있다.

2. Columns of Plots

from bokeh.layouts import column

세로 배열을 해야하는 경우에는 column을 import 한다.

source = ColumnDataSource(flowers)

p1 = figure(plot_width=400, plot_height=400, x_axis_label='Petal Length', y_axis_label='Sepal Length',
           title='Petal Length vs. Sepal Length')
p1.circle('petal_length', 'sepal_length', source=source, color='red')

p2 = figure(plot_width=400, plot_height=400, x_axis_label='Petal Length', y_axis_label='Petal Width',
           title='Petal Length vs. Petal Width')
p2.circle('petal_length', 'petal_width', source=source, color='blue')

layout = column(p1, p2)
show(layout)
Bokeh Plot

row()를 column()으로 변경한 것을 제외하고는 전부 이전과 동일하다.

3. Nested Layouts

source = ColumnDataSource(flowers)

p1 = figure(plot_width=400, plot_height=400, x_axis_label='Petal Length', y_axis_label='Sepal Length',
           title='Petal Length vs. Sepal Length')
p1.circle('petal_length', 'sepal_length', source=source, color='red')

p2 = figure(plot_width=400, plot_height=400, x_axis_label='Petal Length', y_axis_label='Petal Width',
           title='Petal Length vs. Petal Width')
p2.circle('petal_length', 'petal_width', source=source, color='blue')

p3 = figure(plot_width=400, plot_height=400, x_axis_label='Sepal Width', y_axis_label='Sepal Length',
           title='Sepal Width vs. Sepal Length')
p3.circle('sepal_width', 'sepal_length', source=source, color='green')

p4 = figure(plot_width=400, plot_height=400, x_axis_label='Sepal Width', y_axis_label='Petal Width',
           title='Sepal Width vs. Petal Width')
p4.circle('sepal_width', 'petal_width', source=source, color='gold')

layout = column(row(p1, p2), row(p3, p4))
show(layout)
Bokeh Plot

가로, 세로 모두 plot을 배치하고 싶은 경우에는 크게 두 가지 방법을 활용할 수 있다. 첫 번째는 앞서 배운 row()와 column()을 활용하는 것이다. column(row(), row())와 같이 column 안에 row를 삽입하거나, row(column(), column())와 같이 row 안에 column을 삽입하는 등, 어느 방법을 사용하더라도 동일한 결과를 얻을 수 있다.

4. Gridplots

from bokeh.layouts import gridplot

두 번째 방법은 Gridplot 기능을 활용하는 것이다. 우선 bokeh.layouts로부터 gridplot을 import 해야한다.

source = ColumnDataSource(flowers)

p1 = figure(plot_width=400, plot_height=400, x_axis_label='Petal Length', y_axis_label='Sepal Length',
           title='Petal Length vs. Sepal Length')
p1.circle('petal_length', 'sepal_length', source=source, color='red')

p2 = figure(plot_width=400, plot_height=400, x_axis_label='Petal Length', y_axis_label='Petal Width',
           title='Petal Length vs. Petal Width')
p2.circle('petal_length', 'petal_width', source=source, color='blue')

p3 = figure(plot_width=400, plot_height=400, x_axis_label='Sepal Width', y_axis_label='Sepal Length',
           title='Sepal Width vs. Sepal Length')
p3.circle('sepal_width', 'sepal_length', source=source, color='green')

p4 = figure(plot_width=400, plot_height=400, x_axis_label='Sepal Width', y_axis_label='Petal Width',
           title='Sepal Width vs. Petal Width')
p4.circle('sepal_width', 'petal_width', source=source, color='gold')

layout = gridplot([[p1, p2], [p3, p4]])
show(layout)
Bokeh Plot

row나 column을 사용하지 않고, gridplot() 안에 ([ ], [ ], …, [ ]) 형태로 plot들을 추가해주면 된다. 이때 [ ] 안에 있는 plot들끼리 한 행을 이루게 된다.

5. Linking Axes

source = ColumnDataSource(flowers)

p1 = figure(plot_width=400, plot_height=400, x_axis_label='Petal Length', y_axis_label='Sepal Length',
           title='Petal Length vs. Sepal Length')
p1.circle('petal_length', 'sepal_length', source=source, color='red')

p2 = figure(plot_width=400, plot_height=400, x_axis_label='Petal Length', y_axis_label='Petal Width',
           title='Petal Length vs. Petal Width')
p2.circle('petal_length', 'petal_width', source=source, color='blue')

p3 = figure(plot_width=400, plot_height=400, x_axis_label='Sepal Length', y_axis_label='Sepal Width',
           title='Sepal Length vs. Sepal Width')
p3.circle('sepal_length', 'sepal_width', source=source, color='green')

p4 = figure(plot_width=400, plot_height=400, x_axis_label='Sepal Width', y_axis_label='Petal Width',
           title='Sepal Width vs. Petal Width')
p4.circle('sepal_width', 'petal_width', source=source, color='gold')
p2.x_range = p1.x_range
p3.y_range = p1.y_range
p4.x_range = p3.x_range
p4.y_range = p2.y_range

layout = gridplot([[p1, p2], [p3, p4]])
show(layout)
Bokeh Plot

Bokeh가 Interactivity를 지원하는 모듈인 만큼, 단순 배열을 넘어서서 한 화면에 있는 plot들이 동일한 축의 움직임을 공유하도록 설정할 수 있다. a.x_range = b.x_range 혹은 a.y_range = b.y_range와 같이 축 공유를 희망하는 plot들에 대해 x_range와 y_range를 같게 하는 명령어를 사용하면 된다.

축의 공유 뿐만 아니라, Selection Tool을 활용할 때 한 plot에서 선택된 부분이 다른 plot들에서도 선택되게 만들어줄 수 있다. 하지만 이 기능은 별도의 코드를 추가하는 것이 아닌, Gridplot을 사용하여 plot들을 배치한 상태에서 ‘box_select’ 툴을 사용하는 것만으로 실행이 가능하다.

3. Tabbed Layouts

from bokeh.models.widgets import Panel, Tabs

p3.circle('sepal_length', 'sepal_width', source=source, color='green')

p4 = figure(plot_width=400, plot_height=400, x_axis_label='Sepal Width', y_axis_label='Petal Width',
           title='Sepal Width vs. Petal Width')
p4.circle('sepal_width', 'petal_width', source=source, color='gold')
tab1 = Panel(child=p1, title='Tab 1')
tab2 = Panel(child=p2, title='Tab 2')
tab3 = Panel(child=row(p3, p4), title='Tab 3')

tabs = Tabs(tabs=[tab1, tab2, tab3])
show(tabs)
Bokeh Plot

우선 Panel 명령어를 사용하여 각각의 Tab을 형성해준다. Panel() 안에서 child 옵션에 넣고자 하는 plot을, title에 Tab의 제목을 입력해주면 된다. Tab을 다 만들었다면, Tabs() 안에 만들어진 Tab들을 추가해줌으로써 결과물을 완성할 수 있다.
